# 성능/번들 경고 후속 작업 (파일럿 이후)

작성일: 2026-04-17  
상태: 파일럿에는 영향 없음(후속 최적화 대상으로만 기록)

---

## 1) 현재 상황
- `partner-console` 빌드 시 청크 사이즈 경고(예: 500KB 이상)가 발생할 수 있음
- 기능 동작/파일럿 진행에는 직접적인 차질 없음(UX 저하가 체감되면 우선순위 상향)

---

## 2) 후속 작업 후보(우선순위 낮음)
- 라우트/패널 단위 코드 스플릿(`TimelinePanel`, `DraftFilingPanel` 등)
- 다운로드/ZIP 관련 의존성 로딩 시점 최적화(필요 시에만)
- 중복 유틸 제거 및 공통 모듈 정리
