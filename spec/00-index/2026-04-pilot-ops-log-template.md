# 파일럿 일일 운영 로그 템플릿

목적: 파일럿 중 품질/흐름/저장/오류/처리시간을 매일 1회 고정 포맷으로 기록하고, Top3 이슈를 누적하여 에스컬레이션/백로그로 연결한다.

---

## 사용 방법
- 매일 1회, 아래 “일일 로그(복붙용)” 섹션을 복사해 한 블록을 추가한다
- 수치가 없으면 “-”로 두고, 이슈는 반드시 Sev/오너/ETA를 채운다
- 증거 스크린샷은 파일명 또는 링크로 남긴다(가능하면 Partner Console 화면 + caseId 포함)
- 운영 루프 완료조건(1주): “일일 로그(복붙용)” 블록을 5영업일 누적 + Top3 이슈 표의 Sev/오너/ETA 공란 0개
